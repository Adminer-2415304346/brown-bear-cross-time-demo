# Brown Bear Cross-Time Recognition Demo

一个基于 Vue 3 + Vite + Ant Design Vue 的科研演示前端项目，用于展示“跨时间棕熊个体识别”任务，包括数据、模型、系统检索结果与案例演示。

## 1. 项目简介

本项目是一个面向演示视频和现场展示的前端系统，重点围绕以下主线展开：

- 数据集介绍
- 预处理流程
- 模型方案
- 系统检索演示
- 跨时间案例展示
- 指标结果与失败案例

当前项目使用：

- Vue 3
- Vite
- JavaScript
- Ant Design Vue
